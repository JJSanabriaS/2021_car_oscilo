# Engine Diagnostics Device

C-Based Firmware with Full Hardware Integration (NDA Protected)

## Project Overview

This project involves the development of a custom engine diagnostics device designed for real-time vehicle data monitoring, fault detection, and performance analysis. The entire firmware was developed in C, ensuring high efficiency, low-level hardware control, and optimized performance for embedded systems.

Note: Specific technical details are protected under a Non-Disclosure Agreement (NDA).

## Key Features

    * Real-time engine parameter monitoring
    * Diagnostic Trouble Code (DTC) detection and reporting
    * Direct communication with vehicle ECU
    * High-speed data acquisition and processing
    * Secure and stable embedded firmware architecture
